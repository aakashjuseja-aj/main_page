Projects

HAYSTAC - Hidden Activity Signal and Trajectory Anomaly Characterization: Developed ML based models to characterize and detect anomalous human movements in trajectory data ensuring privacy within IoT and Smart City infrastructures. (Funded by IARPA)

Sparse Points to Dense Clouds: Pioneered a 3D detection method leveraging sparse LiDAR data to reconstruct dense point clouds significantly improving detection accuracy on the KITTI and JackRabbot datasets.

Self-Supervised 3D Multiple Object Tracking: Developed the first self-supervised method for 3D object tracking using LiDAR enhancing feature learning and object association without the need for annotated training data. (Funded by Lockheed Martin)

PC-DAN for 3D Multiple Object Tracking: Introduced a novel deep learning model for 3D Multi-Object Tracking that processes sparse LiDAR data with a unique feature permutation approach achieving top leaderboard performance at a Stanford University CVPR workshop. (Funded by Lockheed Martin)

Satellite Image Segmentation: Improved remote sensing image segmentation by modifying U-Net and PSPNet models effectively classifying roads, buildings, grass, and water with enhanced accuracy.

Real-Time Visual Saliency Detection: Proposed a CNN based model for saliency detection in SAR images reducing computational complexity and enabling real-time remote sensing applications. (Funded by Shanghai Jiaotong University, China)
